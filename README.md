# lstree

list directory contents in a tree structure

    testDirectory
    ├──level1
    │  ├──level11
    │  │  ├──a.js
    │  │  └──e.js
    │  └──level12
    │     ├──b.js
    │     └──f.js
    └──level2
       ├──c.js
       └──d.js


## Install

- `npm install -g lstree`
- `lstree -h`

## Usage

- `ls`
- `ls -o <outfile-path>`
- `ls -d <directory-path> -o <outfile-path>`
